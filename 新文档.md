#### 1.4变量数据类型
在JS中一共有六种数据类型：

基本数据类型（值类型）：String 字符串、Number 数值、Boolean 布尔值、Null 空值、Undefined 未定义。

引用数据类型（引用类型）：Object 对象。
**基本数据和引用数据最大的区别**
+ 基本数据类型：参数赋值的时候，传数值。
 	1. 不可改变的原始值：**已经放进去的值不可改变，只会改房间编号为 null(硬盘原理)**
 	```
	var a =10;
	var b = a;
	a = 20;
	console.log(a,b); //20,10
	```
	原始值是我把一个值放到另一个值里面，改了第一个值，第二个值不变.
+ 引用数据类型：参数赋值的时候，传地址（修改的同一片内存空间）。

## 2基本数据类型
#### 2.1 typeof
来检测给定变量的数据类型返回某个字符串.
返回6个值
1. "undefined"，未定义 
2. "boolean" ，布尔
3. "string" ， 字符串
4. "number"	数字
5. "object" 对象或 **null** 
6. "function" 函数。

#### 2.2 undefined类型
使用 var 声明变量但未对其加以初始化时，这个变量的值就是 undefined 
但是包含 undefined 值的变量与尚未定义的变量还是不一样的。
```
var message; // 这个变量声明之后默认取得了 undefined 值
// 下面这个变量并没有声明
// var age
alert(message); // "undefined"
alert(age); // 产生错误
alert(typeof message); // "undefined"
alert(typeof age); // "undefined"
```
#### 2.3 Null类型
null 值表示一个空对象指针,变量的初始值最好初始化为null
实际上， undefined 值是派生自 null 值
```
alert(null == undefined); //true
alert(null === undefined); //false
```
#### 2.4 布尔类型
Boolean只有两个字面值： true 和 false 
Boolean 类型的字面值 true 和 false 是区分大小写的，True 和 False都不是 Boolean 值
要将一个值转换为其对应的 Boolean 值，可以调用转型函数 Boolean()
**转换规则**
+ 转为false的值：""(空字符串)，0,null,NaN,undefined
+ 其余转化为true

#### 2.5 Number类型
Number 类型表示整数和浮点数值,最基本是10进制，支持8进制，16进制。
八进制字面值的第一位必须是零（0），然后是八进制数字序列（0～7），如果超过范围前导零将被忽略，按十进制解析。八进制字面量在严格模式下是无效的。
十六进制字面值的前两位必须是 0x，后跟任何十六进制数字（0～9 及 A～F）。其中，字母 A～F可以大写，也可以小写
###### 2.5.1 数值范围
+ 最小数值保存在 Number.MIN_VALUE 中——在大多数浏览器中，这个值是 5e-324
+ 最大数值保存在Number.MAX_VALUE 中——这个值是 1.7976931348623157e+308
+ 超过这个范围转为Infinity
+ isFinite() 函数。这个函数在参数位于最小与最大数值之间时会返回 true .

###### 2.5.2 NaN
+ 这个数值用于表示一个本来要返回数值的操作数未返回数值的情况.
+ 任何涉及 NaN 的操作都会返回 NaN.
+ NaN 与任何值都不相等，包括 NaN 本身。
+ isNaN() 函数。这个函数接受一个参数，该参数可以是任何类型，而函数会帮我们确定这个参数是否“不是数值”。任何不能被转换为数值的值都会导致这个函数返回 true 

###### 2.5.3 数值转换
Number() 可以用于任何数据类型
parseInt() 和 parseFloat()专门用于把字符串转换成数值
**Number()转化规则：**
+ 如果是 Boolean 值， true 和 false 将分别被转换为 1 和 0。
+ 如果是数字值，只是简单的传入和返回。如果是 null 值，返回 0。
+ 如果是 undefined ，返回 NaN
+ 如果字符串中只包含数字（包括前面带正号或负号的情况），则将其转换为十进制数（前导0会忽略也就是无法使用八进制）
+ 如果字符串中包含有效的十六进制格式，例如 "0xf" ，则将其转换为相同大小的十进制整数值；
+ 如果字符串是空的则将其转换为 0；
+ 如果字符串中包含除上述格式之外的字符，则将其转换为 NaN 。
+ 如果是对象，则调用对象的 valueOf() 方法，然后依照前面的规则转换返回的值。如果转换的结果是 NaN ，则调用对象的 toString() 方法，然后再次依照前面的规则转换返回的字符串值。

**parseInt()规则**
+ 在转换字符串时会忽略字符串前面的空格，直至找到第一个非空格字符。如果第一个字符不是数字字符或者负号， parseInt()就会返回 NaN （**用 parseInt() 转换空字符串会返回 NaN** ）
+ 如果第一个字符是数字字符，parseInt() 会继续解析第二个字符，直到解析完所有后续字符或者遇到了一个非数字字符
+  parseInt() 也能够识别出各种整数格式，但是为了避免识别歧义最好加上第二个参数转换时使用的基数（即多少进制）。

**parseFloat()规则**
+ 与parseInt()类似
+ 两个区别字符串中的第一个小数点是有效的，而第二个小数点就是无效的
+ 始终都会忽略前导0，十六进制格式的字符串则始终会被转换成 0。

#### 2.6 String类型

## 3 js运算符
#### 3.1 一元操作符
###### 3.1.1 递增和递减操作符
	递增和递减操作符分为前置型和后置型。
	```
	var age = 29;
	var anotherAge = --age + 2;
	alert(age); // 输出 28
	alert(anotherAge); // 输出 30
	var num1 = 2;
	var num2 = 20;
	var num3 = (num1--) + num2; //  等于 22
	var num4 = num1 + num2; // 等于 21
	```
	**执行前置递增和递减操作时，变量的值都是在语句被求值以前改变的。
	后置递增，递减操作是在包含它们的语句被求值之后才执行的。**
	4 个操作符对任何值都适用，也就是它们不仅适用于整数，还可以用于字符串、布尔值、浮点数值和对象。对不是数字类型的会调用隐式类型转换变成数字类型，不能转变的为NaN

###### 3.1.2 +，-，*，/，%操作符
+ +运算符
	1. 数字之间是数学运算
	2. 会把不是数字类型隐式调用数字再相加
	3. 任何数据类型加字符串都等于字符串
+ -，*，/运算符
	1. 基本数学运算
	2. 会把不是数字类型隐式调用成数字再运算
+ % 摩尔，模，是取余数
	1. 如果操作数都是数值，执行常规的除法计算，返回除得的余数；
	2. 如果被除数是无穷大值或除数是0,则结果是 NaN ；
	3. 如果被除数是有限大的数值而除数是无穷大的数值，则结果是被除数；
	4. 如果被除数是零，则结果是零；
	5. 如果是 Infinity 被 Infinity 除，则结果是 NaN 

**总结**
1. Nan 加减乘除% 任何数都是NaN  
2. 任何数据类型和undefined运算都是NaN;
3. 任何值和null运算，null可看做0运算。
4. Infinity 加减乘任何数 得到infintiy，除了乘0得到 NaN  
5. infinity /，% infinity 得到NaN
6. 任何数/infinity 得到infinity 除了 0,null 得到 0 
7. Infinity/任何数  得到0 除了0，null 得到 infinity 
8. Infinity % 任何数 得到 NaN
9. 任何数 % infinity 得到 本身  除了 0，null 得到 0 

#### 3.2 布尔操作符
非（NOT）、与（AND）和或（OR）。
+ 非：一个叹号（！）表示，可以应用任何值,这个操作符都会返回一个布尔值.同时使用两个逻辑非操作符，实际上就会模拟 Boolean() 转型函数

###### 3.2.1 与
与操作符由两个和号（ && ）表示
逻辑与操作可以应用于任何类型的操作数，而不仅仅是布尔值。
规则如下：
+ 如果第一个操作数是对象，则返回第二个操作数；
+ 如果第二个操作数是对象，则只有在第一个操作数的求值结果为 true 的情况下才会返回该对象；
+ 如果两个操作数都是对象，则返回第二个操作数
+ 如果有一个操作数是 null/NaN/undefined ，则返回 null/NaN/undefined ；
逻辑与操作属于短路操作，如果第一个操作数是 false ，则无论第二个操作数是什么值，结果都不再可能是true 了。

###### 3.2.2 或
由两个竖线符号（ || ）表示规则：
+ 如果第一个操作数是对象，则返回第一个操作数；
+ 如果第一个操作数的求值结果为 false ，则返回第二个操作数；
+ 如果两个操作数都是对象，则返回第一个操作数；
+ 如果两个操作数是 null/NaN/undefined ，则返回 null/NaN/undefined ；

逻辑或操作符也是短路操作符。也就是说，如果第一个操作数的求值结果为true ，就不会对第二个操作数求值了。
**最大作用**： 利用逻辑或的这一行为来避免为变量赋 null 或 undefined 值。
    `var myObject = preferredObject || backupObject;`

#### 3.3 比较操作符
“>”，”<”，”==”，“>=”，“<=”，”!=” ’===‘ ’！==‘比较结果为 boolean 值
+ 如果两个是数字类型或布尔值，隐式转化成数字按数字大小在比较
+ 如果有一个是字符串，可以转成数字类型按数字大小在比较
+ 不能转成数字类型，字符串转为NaN,结果返回fasle
+ 如果两个都是字符串，比的是 ASCII 码，先比第一位，相同在比较下一位。

**牢记**
NaN 不等于任何数包括自己
null == undefined  // true
null === undefined // false
