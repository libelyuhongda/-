# 基本语法
## 1 变量
#### 1.1变量声明
	```var a //	单个
		var a,b,c,d // 定义多个
	```

#### 1.2变量规则
**起变量名一定要以英文语义化**
1.变量名必须以英文字母、_、$ 开头
2.变量名可以包括英文字母、_、$、数字
3.不可以用系统的关键字、保留字作为变量名
4.变量名长度不能超过255个字符。
5.建议用小驼峰命名规则：

#### 1.3变量赋值
等号表示赋值，将等号右边的值，赋给左边的变量。例 
	`var a = 10`

#### 1.4变量数据类型
在JS中一共有六种数据类型：

基本数据类型（值类型）：String 字符串、Number 数值、Boolean 布尔值、Null 空值、Undefined 未定义。

引用数据类型（引用类型）：Object 对象。
